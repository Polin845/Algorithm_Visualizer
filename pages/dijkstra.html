How Dijkstra's Algorithm Works
  1. Initialize: Set distance to source = 0, all others = ∞
  2. Select: Pick unvisited vertex with smallest distance
  3. Relax: Update distances to neighbors if shorter path found
  4. Repeat: Until all vertices processed
Tip: Green edges show the shortest path tree
