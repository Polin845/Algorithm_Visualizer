How Ford-Fulkerson Works
  1. Find augmenting path: Find a path from source (0) to sink in residual network
  2. Find bottleneck: Find minimum residual capacity along the path
  3. Update flows: Add flow on forward edges, subtract on backward edges
  4. Repeat: Until no augmenting path exists
Tip: Source is node 0 (green), Sink is last node (red)
