What you’re seeing

  1. Counting: for each input value x, increment count[x].
  2. Prefix sums: transform counts so count[i] becomes “how many elements are ≤ i”.
  3. Build output: scan input right-to-left, place each value in its final position using count[value] - 1, then decrement count[value].
Tip: try a slower speed first, then increase it once the idea clicks.
